# 它有什么用?
  我想你一定用过nyaa去下载同人志
  下载过程如下:
  点开链接->查看图片(经过层层广告)->决定是否下载
  使用它后会变成这样:
  查看图片->决定是否下载
  
 # 如何实现
  目前受官方信任的发布者都会使用它[图床](https://hentai-covers.site/)
  通过该图床的标题直接到nyaa获取下载地址
