# 它有什么用?
  我想你一定去nyaa下载过同人志
  它的下载过程十分繁琐:
  点开链接->查看图片(经过层层广告)->决定是否下载
  精简后:
  查看图片->决定是否下载
  
 # 如何实现
  目前受官方信任的发布者都会使用它[图床](https://hentai-covers.site/)来发布图片
  只需在图床内加入下载功能即可,不需要经过任何跳转
